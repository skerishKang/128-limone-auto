## CPU 오류 완화를 위한 BIOS/UEFI 설정 요약

이 문서는 CPU 오류 발생 시 BIOS/UEFI 설정을 통해 시스템 안정성을 높이는 방법을 안내합니다.

**주요 설정 변경 내용:**

*   **CPU 성능 및 전압:**
    *   AMD Ryzen CPU의 경우 "Precision Boost Overdrive(PBO)"와 "Core Performance Boost"를 비활성화하여 온도 안정성을 확보합니다.
    *   "Cool'n'Quiet" 기능을 활성화합니다.
*   **메모리 설정:**
    *   메모리 클럭, 전압을 자동으로 설정합니다.
    *   XMP/DOCP 프로필은 비활성화하여 안정성을 우선합니다.
*   **시스템 안정화:**
    *   "Fast Boot"와 "Above 4G Decoding" 기능을 비활성화합니다.
    *   "Virtualization" 기능은 필요한 경우에만 활성화합니다.
*   **온도 관리:**
    *   팬 속도 설정을 "Performance" 또는 "Turbo"로 변경하여 냉각 성능을 강화합니다.
    *   CPU 온도 경고 설정은 85°C로 지정하고, 시스템 온도 모니터링을 활성화합니다.

**실행 순서:**

1.  **중요 데이터 백업**
2.  **BIOS 설정 변경**
3.  **Windows에서 안정성 테스트**
4.  **문제 지속 시 전문가 상담**

**핵심 목표:**

CPU의 과도한 성능 향상 기능을 제한하고, 메모리 설정을 안정화하며, 불필요한 기능을 비활성화하고, 냉각 성능을 강화함으로써 CPU 오류를 완화하고 시스템 안정성을 높이는 것입니다.
